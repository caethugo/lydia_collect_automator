What is this bot ?

Goal behind this bot

The objective of this project is to automate the shotgun of events'places.
It works especially for events that have a facebook page dedicated and for shotguns organized with a Lydia collect. It can ensure one's presence to any event organized with Lydia on Facebook.

How to use the bot

The bot will be taking as input :

  • The facebook page link for the event
  • The user's facebook credentials in order to access the page *
  • The number of fields he wants to complete

Then, the user will be asked to enter his key / value fields according to the number of fields he provided.
For instance, if you know there will be a "Name" field in the Lydia collect, you have to provide the field Name with the value being your name. taking a URL and a list of attributes as input One you gave all your key / value pairs, the bot will be able to detect on the facebook page you provided the presence of a lydia collect link. Eventually, it will open it, and complete the form with the given attributes.

The user can activate 3 toggles :
  • Auto submit form
    • The bot can either let the user check the form content before submitting it himself, or activate this toggle to let the bot submit the form right after completing all the fields.
  • Accept Cookies
    • The user can either accept facebook cookies manually or let the bot accept those.
  • Enter credentials manually
    • If the user wants to enter his credentials himself and not give them to the website, he can. Once the bot is launched, the user will just have to input it in the facebook page opened by the bot.
